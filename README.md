# FactoorSharp

## Introduction
The FactoorSharp FacturX component allows to create XML files as required by German electronic invoice initiative ZUGFeRD as well invoices in the successor Factur-X. 
The component supports XRechnung which is a special profile in the German ZUGFeRD format. XRechnung can be expressed either in CII (which is the format used for ZUGFeRD) and UBL (as defined by OASIS and used by Peppol).

The component is meant to create and read electronic invoices as easy as possible.
